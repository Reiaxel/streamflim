# Streamflim - Clon de Netflix

**Streamflim** es una aplicación inspirada en Netflix, desarrollada con Next.js y TypeScript. Permite explorar diferentes secciones de contenido como series, documentales y películas, con una experiencia visual atractiva y un sistema interactivo de trailers al pasar el mouse sobre una imagen.

---

## Tecnologías utilizadas

- Next.js (React Framework)
- TypeScript
- Tailwind CSS para estilos modernos y responsivos
- API externa para obtener trailers de películas y series
- Context API para manejo de estado (si aplica)

---

## Funcionalidades principales

- Navegación por secciones: películas, series, documentales
- Reproducción de trailers al pasar el mouse por las imágenes
- Diseño visual atractivo y responsivo
- Interfaz clara y amigable
